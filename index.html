Project Documentation

A clean, fast, and simple documentation site for your GitHub Pages project. Everything you need to get started in one place.

Getting started

Welcome! This template is designed to be a drop-in documentation page for any open-source project. It is built with vanilla HTML and Tailwind CSS, so it works out of the box on GitHub Pages.

Tip: Edit the sections below directly in index.html to match your project.

Installation

Install the package using your favorite package manager.

# npm
npm install your-package

# yarn
yarn add your-package

# pnpm
pnpm add your-package

Configuration

Create a config file in the root of your project.

// docs.config.js
export default {
  title: 'My Project',
  description: 'A short description',
  theme: 'system',
};

Quick start

Import and initialize the library in your application entry point.

import { createApp } from 'your-package';

const app = createApp({
  apiKey: 'your-api-key',
  debug: true,
});

app.start();

FAQ

Can I customize the theme?

Yes. The page uses Tailwind CSS classes and respects the user's system color scheme. You can override styles in the inline <style> block.

Is search enabled?

The header includes a search button as a placeholder. You can wire it to a search index or a third-party service like Algolia DocSearch.

Troubleshooting

If something isn't working, try these steps:

  1. Check that you are on the latest version.
  2. Clear your browser cache and reload.
  3. Review the console for error messages.
  4. Open an issue on GitHub.
